# Workout Images Directory

This directory contains images for workout cards in the Neurofit app. The images should be high-quality, properly compressed, and follow these guidelines:

- Resolution: 1920x1080 (16:9 aspect ratio)
- Format: JPG or WebP
- File size: Optimized, preferably under 200KB
- Naming convention: kebab-case (e.g., hiit-cardio.jpg)

## Image Credits
Please ensure all images used are properly licensed and credited.
