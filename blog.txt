Firebase Blogs Collection Structure:

Collection: 'blogs'
  Document: {auto-generated-id}
    Fields:
      - title: string
      - summary: string
      - content: {
          introduction: string
          sections: [
            {
              type: "heading" | "paragraph"
              content: string
            }
          ]
          conclusion: string
        }
      - image: string (URL)
      - category: string
      - tags: string[]
      - createdAt: timestamp
      - updatedAt: timestamp

Query Examples:
1. Get all blogs:
   - collection('blogs').get()

2. Get blog by ID:
   - collection('blogs').doc('abc123def456').get()

3. Get blogs by category:
   - collection('blogs').where('category', '==', 'Fitness').get()

4. Get latest blogs:
   - collection('blogs').orderBy('createdAt', 'desc').limit(10).get()

5. Get blogs with specific tag:
   - collection('blogs').where('tags', 'array-contains', 'workout').get()

Note: All content fields (introduction, sections content, conclusion) are stored as plain text without HTML tags.
